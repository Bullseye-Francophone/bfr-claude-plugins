# BFR Claude Code plugins

Claude Code plugin marketplace by Bullseye Francophone (BFR), for DCS World mission makers.

## Plugins

- **dcs-mission-tools** — understand, lint and explain DCS missions built on the VEAF mission template. See `plugins/dcs-mission-tools/README.md`.

## Install

```
/plugin marketplace add Bullseye-Francophone/bfr-claude-plugins
/plugin install dcs-mission-tools@bfr
```
